JAVA is a HHL (high level langauge) and platform independent langauge also because when you write Java
code, it's ultimately written in JVM not on your physical machine. Since JVM executes the Java bytecode
which platform independent.

1. JVM(java virtual machine) : JVM is that machine which enable your computer to run a java program. 
                               when developer run the java program the java compiler will compile the 
                               java code to bytecode. and JVM translate bytecode into the machine code.

    ex.

    [java program] --------java compiler-------> [java bytecode] --------JVM--------> [machine code] ---------CPU----->[output]

**********************************************************************************************************

2. JRE(java runtime environment) : JRE provides java class libraries, JVM and other components that are require
                                   to run java application.

**********************************************************************************************************

3. JDK (java development kit) : JDK require to develop application in java when you download JDK & JRE is also download with it.
